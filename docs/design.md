# 霓虹弹球打砖块设计文档

## 1. 产品定位

霓虹弹球打砖块是一款可在浏览器中即时游玩的平面 2D 弹球打砖块游戏。项目目标是保持经典打砖块游戏的核心手感，同时通过霓虹视觉、粒子反馈、连击倍率、核心道具和关卡模式增强现代游戏感。

## 2. 设计目标

- 纯前端、无后端、无付费素材，适合 EdgeOne Pages 静态部署。
- 鼠标、触屏、键盘均可操作。
- HUD 信息一眼可读，不遮挡主要游戏区域，并在 PC、移动端、短屏横屏和 ≤360px 窄屏下保持可读。
- 游戏反馈明确：击中、爆炸、获得道具、掉球、通关、限时失败和首领技能都有视觉和声音反馈。
- 通过关卡系统 2.0 提供常规、限时、首领三类节奏，兼顾新手引导和进阶挑战。

## 3. 视觉风格

整体采用平面 2D 霓虹街机风格：

- 背景：深蓝到紫黑渐变，叠加低透明度网格。
- 球体：白色高亮核心，蓝色或火球红色光晕。
- 砖块：高饱和霓虹色块，带顶部高光和底部暗面；高密度小砖块会自适应圆角、血条和标识字号。
- 挡板：玻璃胶囊形态，蓝粉渐变边缘。
- 道具：圆形能量胶囊，使用中文单字标识。
- HUD：半透明玻璃面板，展示分数、关卡、模式、计时和机会；数字值在极窄空间内省略防溢出。
- 关卡选择：玻璃卡片式列表，显示模式、难度、限时或首领脉冲信息；长标题和描述允许换行，避免窄屏裁切。

## 5. 页面和状态

### 5.1 开始页

包含：

- 游戏标题（PC 与移动端保持单行）
- 简短玩法说明
- 开始游戏按钮
- 关卡选择卡片列表
- 道具图标列表（仅图标+名称，简洁展示4种道具）
- 操作提示

### 5.2 游戏中 HUD

HUD：

- 分数：当前分数
- 最高分：本地最高分
- 关卡：当前关卡 / 总关卡
- 模式：当前模式，常规 / 限时 / 首领
- 时间：限时关倒计时、首领技能倒计时或无限符号
- 机会：当前关剩余机会
- 暂停：暂停按钮

底部道具栏：

- 多球
- 加宽
- 火球
- 护盾

### 5.3 暂停页

包含：

- 继续
- 重新开始
- 选择关卡
- 道具说明区域（完整描述4种道具的效果和持续时间）

### 5.4 下一关过渡页

当玩家清空当前关卡且不是最终关时，会先进入下一关过渡页，而不是立刻切换关卡。

包含：

- `关卡完成` 状态提示
- `确认进入第 N 关？` 确认提示
- 下一关名称
- 下一关模式和描述
- 本关奖励分数
- 确认进入下一关按钮
- 选择关卡按钮

### 5.5 结算页

包含：

- 胜利 / 游戏结束
- 得分
- 最高分
- 到达关卡
- 当前模式
- 最大连击
- 重新开始
- 选择关卡

## 6. 游戏规则

### 6.1 基础规则

| 规则 | 说明 |
|---|---|
| 目标 | 消除当前关卡所有可消除砖块 |
| 操作 | 鼠标、触屏、方向键控制挡板 |
| 反弹 | 球撞墙、挡板、砖块时反弹；多球状态下每颗球每帧最多处理一次砖块碰撞，保证多球清场反馈 |
| 掉球 | 球落出底部失去一次机会 |
| 机会 | 简单关卡 1 次机会，较难关卡 2 次机会 |
| 胜利 | 清完 16 个关卡 |
| 失败 | 机会归零，或限时关倒计时归零 |

### 6.2 关卡模式

| 模式 | 规则 | HUD 表现 |
|---|---|---|
| 常规 | 常规清砖节奏，无倒计时 | 时间显示 `∞` |
| 限时 | 必须在限制时间内清空可消除砖块 | 时间显示剩余秒数 |
| 首领 | 首领砖块血量更高，并周期性释放首领脉冲加速球 | 时间显示下次脉冲倒计时 |

关卡配置支持难度、描述、球速倍率、道具掉落率、限时时间、首领技能间隔和首领血量。当前版本共 16 关，关卡布局提升为 12-14 列高密度矩阵，每关比基础版本增加 1-2 行砖块，并通过更小砖块和更紧凑间距提升清场目标数量。机会数由关卡难度决定：难度 1-4 为 1 次机会，难度 5-9 为 2 次机会。

### 6.4 砖块类型

| 类型 | 规则 | 视觉 |
|---|---|---|
| 普通砖块 | 击中 1 次消除 | 彩色霓虹块 |
| 加固砖块 | 击中 2 次消除 | 带生命条 |
| 钢铁砖块 | 不可消除，只反弹 | 金属灰扫光 |
| 爆炸砖块 | 消除周围非钢铁、非首领砖块 | 红色，显示 `×` |
| 首领砖块 | 血量由关卡配置决定，命中后显示血量提示、粒子和震屏反馈 | 粉色高血量块，带血条和血量数字 |

### 6.5 道具

| 道具 | 效果 |
|---|---|
| 多球 | 分裂为多球 |
| 加宽 | 挡板加长 15 秒 |
| 火球 | 火球穿透砖块 10 秒 |
| 护盾 | 底部护盾抵挡一次掉球；移动端覆盖整条可玩底线，避免边缘掉球未被保护 |

### 6.6 计分

- 普通砖块：按行 10-50 分。
- 加固砖块：普通分值 2 倍。
- 首领砖块：普通分值 3 倍。
- 爆炸砖块：80 分。
- 接住道具：100 分。
- 连击倍率：最高按 x8 结算。
- 单关奖励：剩余机会 ×1000 分。
- 限时奖励：剩余秒数 ×20 分。

## 7. 动效策略

- 球体：拖尾 + 光晕。
- 首领砖块命中：显示血量提示、砖块内血量数字、粒子爆裂和轻震屏。
- 胜利时进入下一关过渡页，显示下一关名称、模式、描述和奖励分数。
- 爆炸砖块：更大粒子量 + 震屏。
- 首领脉冲：震屏 + 提示 + 合成爆炸音效。
- 道具获得：提示 + HUD 激活。
- 掉球：震屏 + 暂停页。

## 8. 可访问性

- Canvas 提供 `aria-label`。
- HUD 使用语义化区域和按钮。
- 关卡选择使用真实按钮，可键盘聚焦。
- 关键状态通过实时提示区域告知玩家。
- 所有按钮支持键盘聚焦。
- CSS 提供 `prefers-reduced-motion` 降低动画。
